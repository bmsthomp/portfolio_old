Projects

Sun Runner


This is a canvas and javascript game that I built for a Rich Internet Applications course while working on my minor in computer science. The game implements a wide range of javascript techniques for game development. Everything was coded from scratch including the game physics and the paralax background. Game control is tight and the physics react smoothly to object collision. Give it a try and see how long you can last!

SugarCRM


While working as a web developer for the University of Wisonsin Milwaukee I worked with a small team of two to implement an instance of SugarCRM for the school. The CRM keeps track of student admission and academic progress for the school. I took lead on the expansion and growth of the project for our school. I also worked closely with another developer on the construction of a student portal for applying to the school. This portal directly interfaced with the CRM to input student information.

Learning Solutions Inc.


Learning Solutions is a fictitious company that I worked with for my capstone course. The goal of this project was to create website that allowed users to sign up and view educational material on the web. I built the site entirely with PHP and used Bootstrap CSS for the visual front end. If you're interested click on the link and take a look! Feel free to create an account.
